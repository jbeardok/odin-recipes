# odin-recipes
This comprehensive project is a recipe website. It includes implementations of previously learned concepts,
such as the htmml boilerplate, descriptive paragraph elements, inline elements (<em>em</em>, <strong>strong</strong>), 
images with alt attributes, ordered lists for recipe steps, unordered lists for recipe ingredients, and links for 
navigating the website. The file structure of this project meets the web development standard. At
the end of the development phase, all files are checked with w3's HTML Validator tool for any potential errors.
